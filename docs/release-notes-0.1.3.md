# Suwol Tools 0.1.3

## 변경 사항

- 사용자가 전달한 공식 Suwol Tools 앱 아이콘으로 교체했습니다.
- Windows, macOS, Linux 패키지가 하나의 원본 아이콘에서 생성된 리소스를 사용합니다.
- Windows NSIS·portable, macOS DMG·ZIP, Linux AppImage·tar.gz의 아이콘 연결과 검증을 보강했습니다.
- 앱 헤더와 웹 favicon에도 공식 아이콘을 적용했습니다.
- 기존 69개 도구의 기능과 자동 업데이트 동작은 변경하지 않았습니다.
- Windows 자동 업데이트는 기존 정책대로 비활성 상태입니다.
