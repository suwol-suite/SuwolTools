## 전환 판정

원본 Web 프로젝트는 수정하지 않는다. Renderer는 공통 입력/출력/작업 큐 UI만 담당하고, 파일을 읽거나 결과를 생성하는 작업은 Worker/child process가 담당한다. 브라우저에서만 가능한 preview는 Electron에서 proxy handle 또는 안전한 IPC로 제한하며, 실제 결과 생성과 저장은 같은 `InputSource`, `OutputTarget`, `Job`, `JobResult` 계약을 사용한다.

7개는 공통 feature shell, Worker/child process/IPC, 회귀 테스트를 통과해 `migrated: true`로 승격했다. PDF encryption은 실제 cross-platform adapter가 없으므로 UI에 노출하지 않는다. 배포판에서는 검증된 FFmpeg resource를 공급해야 미디어 결과 작업이 활성화된다.
